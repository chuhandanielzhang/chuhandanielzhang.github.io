Course Projects at GTIIT

Academic Projects: Advanced Robotics Laboratory & Design for Manufacturing

TurtleBot Autonomous Navigation

Course: Advanced Robotics Laboratory (034401), Prof. Damiano Padovani

Developed a complete autonomous navigation and obstacle avoidance system for TurtleBot 4 using ROS2.

Technical Implementation

  • ROS2 System Architecture: Modular software stack within ROS2 Humble framework.
  • Motion Control: Python-based nodes publishing Twist messages with closed-loop velocity control.
  • Sensor Integration: Asynchronous processing of OAK-D depth camera and 2D LiDAR data.
  • Autonomous Logic: Finite State Machine transitioning between PATH_TRACKING and OBSTACLE_AVOIDANCE.
  • Path Tracking: PID control loop with visual servoing for line following.
  • Reactive Avoidance: LiDAR-based collision avoidance with circumnavigation maneuvers.

Unitree Z1 Robotic Arm

Course: Advanced Robotics Laboratory (034401), Prof. Damiano Padovani

Developed autonomous trajectory planning and control algorithms for the Unitree Z1, a 6-DOF robotic arm.

Technical Implementation

  • Kinematics: Dual-space control with Inverse Kinematics solvers for Cartesian to Joint-space mapping.
  • Trajectory Planning: 5th-order polynomial generation for C2-continuous motion.
  • Autonomous Manipulation: Pick-and-place logic using MoveJ and MoveL commands.
  • High-Precision Path Tracking: Sub-millimeter repeatability with constant orientation constraints.

Delta Parallel Manipulator

Course: Advanced Robotics Laboratory (034401), Prof. Damiano Padovani

Programmed a GTHZ-300B industrial Delta robot for high-speed pick-and-place automation.

Technical Implementation

  • Kinematics: Analyzed 3-DOF parallel kinematic chain for high-speed translational movement.
  • System Calibration: Calibrated kinematic chain and pneumatic subsystems (4 bar).
  • PTP Programming: Teach-and-Playback methodology for precise Cartesian workspace.
  • Sequential Logic: Optimized AxisHome, MoveDoor, and SetOutputBeforeEnd commands.

Rebound Robotics - Basketball Training Robot

Course: Design for Manufacturing (034371), Dr. Anand Prakash Dwivedi

Led team to design an autonomous basketball training robot with navigation, collection, and passing capabilities.

My Role: Lead CAD Designer & FEA Engineer

Mechanical System Design

  • Omnidirectional Chassis: Four Mecanum Wheels for instant translation and rotation.
  • Collection System: Elastic rollers with motor-driven conveyor.
  • Internal Hopper: 6-ball capacity funnel with optimized guide slopes.
  • Dual Friction-Wheel Launcher: PWM-controlled for adjustable pass velocity.

FEA Analysis

  • Chassis Static Analysis: Safety Factor 2.24, exceeding 1.5 requirement.
  • Chassis Fatigue Analysis: Fatigue safety factor above 1.50 in all critical regions.
  • Launcher Analysis: Maximum displacement less than 0.016 Œºm, life exceeds 1,000,000 cycles.

Thermal Imaging Tracking Gimbal

Integrated computer vision and thermal imaging on NVIDIA Jetson Nano for intelligent tracking gimbal.

Core Contribution

Implemented Bicubic Interpolation algorithm to upsample low-resolution thermal sensor data (AMG8833) into high-resolution thermal maps in real-time.

Core Hardware

  • NVIDIA Jetson Nano
  • Dual-Axis (Pan/Tilt) Stepper Motor Gimbal
  • Webcam + AMG8833 Thermal Imaging Sensor

Features

  • Facial Tracking: OpenCV + Haar-Cascade classifier for real-time face detection and tracking.
  • Meditation Mode: Detects stillness to control ambient lighting.
  • Watch Movie Mode: Monitors movement and adjusts environment accordingly.
