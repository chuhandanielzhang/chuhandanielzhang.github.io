TurtleBot Autonomous Navigation

Advanced Robotics Laboratory (034401), GTIIT, Spring 2025

Instructor: Prof. Damiano Padovani

Developed autonomous navigation and obstacle avoidance routines for TurtleBot 4 using ROS2 Humble framework.

Technical Implementation

  • ROS2 Architecture: Built modular workspace with custom Python nodes for motion control.
  • Teleoperation: Implemented keyboard-based teleop publishing Twist messages to cmd_vel topic.
  • Circular Motion: Programmed closed-loop velocity control for specified radius trajectory (~30 cm).
  • Path Following: Developed timer-based autonomous routines to follow predefined paths.
  • Obstacle Avoidance: Integrated 2D LiDAR for real-time collision detection and circumnavigation.
  • Sensor Fusion: Studied OAK-D depth camera and LiDAR integration for robust navigation.
