Thermal Imaging Tracking Gimbal

Course Project, GTIIT, 2024

Integrated computer vision and thermal imaging on NVIDIA Jetson Nano for intelligent tracking gimbal.

Core Contribution

Implemented Bicubic Interpolation algorithm to upsample low-resolution thermal sensor data (AMG8833) into high-resolution thermal maps in real-time.

Core Hardware

  • NVIDIA Jetson Nano
  • Dual-Axis (Pan/Tilt) Stepper Motor Gimbal
  • Webcam + AMG8833 Thermal Imaging Sensor

Features

  • Facial Tracking: OpenCV + Haar-Cascade classifier for real-time face detection and tracking.
  • Meditation Mode: Detects stillness to control ambient lighting.
  • Watch Movie Mode: Monitors movement and adjusts environment accordingly.
