## 🌐 启用 GitHub Pages

### 步骤：

1. **打开仓库**：
   https://github.com/chuhandanielzhang/chuhandanielzhang.github.io

2. **点击 "Settings"**（顶部导航）

3. **左侧菜单，找到并点击 "Pages"**

4. **向下滚动到 "Build and deployment"**

5. **配置 Source**：
   - 点击下拉菜单
   - 选择：**"Deploy from a branch"**

6. **选择分支**：
   - Branch 下拉菜单选择：**"main"**
   - Folder 自动变成：**"/ (root)"**

7. **点击绿色的 "Save" 按钮**

8. **等待 1-2 分钟**
