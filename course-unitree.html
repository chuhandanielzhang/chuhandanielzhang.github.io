Unitree Z1 Robotic Arm

Advanced Robotics Laboratory, GTIIT, 2024

Course: Advanced Robotics Laboratory (034401), Prof. Damiano Padovani

Developed autonomous trajectory planning and control algorithms for the Unitree Z1, a 6-DOF robotic arm.

Technical Implementation

  • Kinematics: Dual-space control with Inverse Kinematics solvers for Cartesian to Joint-space mapping.
  • Trajectory Planning: 5th-order polynomial generation for C2-continuous motion.
  • Autonomous Manipulation: Pick-and-place logic using MoveJ and MoveL commands.
  • High-Precision Path Tracking: Sub-millimeter repeatability with constant orientation constraints.
