Unitree Z1 Robotic Arm

Advanced Robotics Laboratory (034401), GTIIT, Spring 2025

Instructor: Prof. Damiano Padovani

Programmed trajectory planning and autonomous manipulation for the Unitree Z1, a 6-DOF robotic arm using z1_sdk.

Technical Implementation

  • SDK Integration: Configured z1_controller and z1_sdk for real-time robot communication.
  • Motion Control: Implemented joint-space (JOINTCTRL) and Cartesian-space (CARTESIAN) control modes.
  • Trajectory Planning: Applied 3rd and 5th-order polynomial paths for smooth acceleration profiles.
  • Pick-and-Place: Developed autonomous manipulation cycles using MoveJ, MoveL, and MoveC commands.
  • Precision Drawing: Programmed end-effector path tracking for marker-based drawing tasks.
  • Teach-and-Playback: Utilized TEACH mode for trajectory recording and TEACHREPEAT for replay.
